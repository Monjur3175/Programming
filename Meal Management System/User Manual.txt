Mess Management System User Manual
This manual provides instructions on how to use the Mess Management System. The system allows you to manage members, expenses, meals, and calculate bills for a mess (shared living arrangement where meals are shared).

1. Main Menu:

When you run the program, you will be presented with the main menu:

--- Mess Management System ---
1. Login
2. Register
3. Manage Members
4. Manage Expenses
5. Manage Meals
6. Calculate Bills
7. Exit
Enter your choice:
2. Login/Registration:

1. Login: If you have already registered, enter your username and password. If the credentials are correct, you will see a "Login successful!" message. Currently, there is no further action after login within this version.
2. Register: If you are a new user, select this option. You will be prompted to enter a new username and password. After successful registration, you will see a "Registration successful!" message.
3. Manage Members:

Select option 3 from the main menu to manage members. The following submenu will appear:

--- Member Management ---
1. Add Member
2. View Members
3. Remove Member
4. Back to Main Menu
Enter your choice:
1. Add Member: Enter the member's ID (integer), name (string), and contact information (string). The member's initial meal count will be set to 0.
2. View Members: Displays a list of all members with their ID, name, contact, and meal count.
3. Remove Member: Enter the ID of the member you want to remove. The system will remove the member from the records.
4. Back to Main Menu: Returns you to the main menu.
4. Manage Expenses:

Select option 4 from the main menu to manage expenses. The following submenu will appear:

--- Expense Management ---
1. Add Expense
2. View Expenses
3. Back to Main Menu
Enter your choice:
1. Add Expense: Enter the date (YYYY-MM-DD), a description of the expense, and the amount (double).
2. View Expenses: Displays a list of all expenses with their date, description, and amount.
3. Back to Main Menu: Returns you to the main menu.
5. Manage Meals:

Select option 5 from the main menu to manage meals. The following submenu will appear:

--- Meal Management ---
1. Add Meal
2. View Meals
3. Back to Main Menu
Enter your choice:
1. Add Meal: Enter the member's ID, the date of the meal (YYYY-MM-DD), and the number of meals consumed by that member on that date.
2. View Meals: Displays a list of all meal entries with the member's ID, date, and number of meals.
3. Back to Main Menu: Returns you to the main menu.
6. Calculate Bills:

Select option 6 from the main menu to calculate bills. The system will:

Calculate the total expenses from the recorded expenses.
Calculate the total number of meals from the recorded meal entries.
Calculate the meal rate (total expenses / total meals).
Display the meal rate.
Display each member's bill (member's meal count * meal rate).
If there are no meals or expenses recorded, the system will display an appropriate message.

7. Exit:

Select option 7 from the main menu to exit the program.

Data Storage:

The system stores data in the following text files:

users.txt: Stores user credentials (username and password).
members.txt: Stores member information (ID, name, contact, and meal count).
expenses.txt: Stores expense information (date, description, and amount).
meals.txt: Stores meal information (member ID, date, and number of meals).
Date Format:

Please use the YYYY-MM-DD format for dates (e.g., 2024-10-27).

Error Handling:

The system includes basic error handling for file operations and invalid input.

This manual should help you effectively use the Mess Management System. If you have any further questions, please consult with the developer.
